# GUIA para tradução de python a java

## Imprimir Arrays
 * No livro vai aparecer print(lista), no java tem que colocar System.out.println(Arrays.toString(lista));

## Comparação
 * Para números(int, double), utilizar ==.
 * Para Objetos e Textos(String, Integer) *NUNCA* usar ==, utilizar .equals(). Ex: if (nome.equals("João"))
